主进程和渲染进程

由于 Electron 使用了 Chromium 来展示 web 页面，所以 Chromium 的多进程架构也被使用到。 每个 Electron 中的 web 页面运行在它自己的渲染进程中。

相当于浏览器窗口时主进程，每个tab是渲染进程。

主进程：

  • 使用和系统对接的Electron API, 就是调用系统API;
  • 创建渲染进程；
  • 全面支持nodejs
  • 只有一个，是程序的入口

渲染进程：

  • 多个，单独
  • 支持 nodejs\ dom api
  • 使用部分Electron API
